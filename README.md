# 30 Days of NumPy and Pandas Problems

A focused 30-day practice series to build strong fundamentals in NumPy and Pandas —
the two most essential libraries for Data Science.
Each day covers core problems that sharpen problem-solving skills
and build the kind of muscle memory that matters in real projects and interviews.

---

## Goal

- Strengthen NumPy and Pandas problem-solving from scratch
- Build a daily GitHub contribution habit
- Cover real gaps in Data Science fundamentals

---

## Libraries Used

- Python 3.x
- NumPy
- Pandas
